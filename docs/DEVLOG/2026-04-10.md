# DEVLOG - 2026-04-10

## 앉기(Crouch) 이슈 해결
- 증상: `IA_Crouch` 입력은 들어오나 앉지 않음. 로그에 `CanEverCrouch=0`, `LogCharacter: ... crouching is disabled ... NavAgentSettings`
- 원인: BP(`AgentA`/`AgentB` 등)의 **Character Movement → Nav Agent**에서 **`Can Crouch`**가 꺼져 있어, C++ 생성자의 `NavAgentProps.bCanCrouch`가 인스턴스 기본값에 덮임
- 해결:
  - 런타임: `BaseFPSCharacter::BeginPlay`에서 `NavAgentProps.bCanCrouch = true` 재적용
  - 에디터: 캐릭터 BP에서도 **Nav Agent → Can Crouch** 켜 두면 설정이 명확해짐 (코드와 중복 가능)

## 애니 인스턴스
- `FPSAnimInstance`: `bCrouching`, `bJumping`, `bJumpPressed`, `bEnableJump` 등 정리 (`UKismetAnimationLibrary::CalculateDirection`, `AnimGraphRuntime` 모듈)
